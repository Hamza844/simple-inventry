# Simple Inventory (Node.js + Express + SQLite)
## Description
Simple Inventory REST API with MVC structure. Supports Products and Orders.

## Run
1. `npm install`
2. `node app.js` or `npm start`
3. Server runs on port 3000

## Endpoints
- POST /products
- GET /products
- GET /products/:id
- POST /orders
- GET /orders
